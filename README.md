# Varity App Store

**Live:** [store.varity.so](https://store.varity.so)

The consumer-facing marketplace for discovering and launching apps built on Varity. Browse, search, and open apps — no account required.

## Overview

The App Store is designed for end users who want to find and use business applications. The experience mirrors familiar app stores: browse categories, read descriptions, and launch apps instantly.

- Clean, consumer-friendly interface
- Email and social login (no technical setup)
- One-click app launching
- Category-based discovery

## Development

```bash
npm install
npm run dev
```

Runs on [localhost:3000](http://localhost:3000).
