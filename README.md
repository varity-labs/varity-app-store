## About

The Varity App Store is a consumer marketplace for discovering and launching apps built by developers on the Varity platform. Browse by category, read descriptions, and open apps — no technical setup required.

## Local Development

```bash
npm install
npm run dev
# Open http://localhost:3000
```

```bash
npm run build    # Production build
npm run lint     # Lint
```
